# Full-Stack-Development-Journey
Moving from Basic Web design to a Full Stack development

#########################################################
# Below is a **serious 2-month (8-week) crash plan** to get you to **job-ready full-stack level** fast. It focuses on **practice > theory** and ends with a **portfolio you can show employers**.

I’ll assume you’re starting from basics and can study **3–6 hours/day**.

---

# ⚡ 2-Month Full-Stack Web Development Crash Plan

## Stack you will learn (keep it simple)

* Frontend: HTML, CSS, JavaScript, React
* Backend: Node.js, Express
* Database: MongoDB
* Tools: Git, GitHub, VS Code
